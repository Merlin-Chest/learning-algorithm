# 排序

## 排序模板

为了更好的可读性和可移植性

```js
class Sort{
	static sort(nums){}
	// 比较元素
	static less(num1, num2){}
	// 交换元素
	static exch(nums, i, j){}
	// 打印元素
	static show(nums){
		console.log(nums.join(','));
	}
	// 测试是否已经有序
	static isSorted(nums){
		for(let i = 1; i < nums.length; i++){
			if(less(nums[i], nums[i - 1])) return false;
		}
		return true;
	}
}
```
